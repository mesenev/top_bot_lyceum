# top_bot_lyceum
Телеграм-бот, созданный для поддержки обучающихся программированию.

# Команды
/top - выводит топ-5 студентов группы.

/login - залогиниться

/hw - начать проверку домашних работ (для преподавателей). Перед началом проверки нужно залогиниться.
